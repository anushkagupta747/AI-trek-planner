# 🏔️ AI Trek Planner

A personalized trek planning system that generates packing lists and (future) itineraries based on user inputs like difficulty, duration, season, and trek style.

---

## 🚀 Overview

Planning a trek involves multiple decisions — what to pack, how to prepare, and what to expect.

This project aims to simplify that by building an **intelligent trek planner** that:
- Takes structured trek inputs
- Applies rule-based logic (initially)
- Evolves into an AI-powered assistant

---

## ⚙️ Features (Current)

- ✅ User input system via CLI
- ✅ Structured trek data collection:
  - Trek name
  - Difficulty
  - Duration
  - Month
  - Trek style
- 🚧 Packing list generation (rule-based logic) *(in progress)*

---

## 🧠 Inputs

Example:
Enter trek name: Hampta Pass
Enter difficulty (easy/moderate/hard): moderate
Enter number of days: 5
Enter month of trek: july
Enter trek style (camping/teahouse/day trek): camping
